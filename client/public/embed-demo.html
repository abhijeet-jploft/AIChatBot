Chat widget embed demo

This page loads the widget via the script below. The widget will activate after 6–10 seconds, or 40% scroll, or 8 seconds idle. When you open it, Anaya’s opening message will appear.

Configure data-api-url to your API (e.g. https://your-server.com/api). Ensure the API allows CORS from this origin. 

<script
  src="https://your-domain.com/chat-widget.js"
  data-api-url="https://your-api.com/api"
  data-company-id="_JP_Loft"
  data-company-name="JP Loft"
></script>
